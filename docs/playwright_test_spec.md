# 企業バイアス分析ダッシュボード Playwrightテスト仕様書（Chrome限定）

## 1. テスト対象
- `app.py`（Streamlitダッシュボード 単日分析画面）

## 2. 前提条件
- テスト用データ（corporate_bias_dataset.json等）が正しく配置されている
- Streamlitアプリがローカルまたはテスト環境で起動済み
- Playwrightのセットアップ・依存パッケージインストール済み
- **Chromeブラウザのみでテストを実施**
