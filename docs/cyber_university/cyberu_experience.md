# サイバー大学を社会人学生が卒業してみて思ったこと

## 入学の経緯
- 千葉大工学部を中退し大卒資格がなく、喉に小骨が引っかかったようなモヤモヤが存在したままエンジニアになりました。
- SES でエンジニア → 中華系 SIer に「AIエンジニア」として転職したが、実際は開発リーダーとして顧客折衝メインでした。
- その経験は成長にはつながったが多忙すぎてAIスキルが伸びず、強制力のある学び場としてサイバー大学を選択。

## 学んで良かったこと（メリット）
- 講義は時系列で学びつつ最新の内容まで取り入れており質が高い。
- 体系的に講義を履修しないと先に進めない仕組みもあり、自分の方向性を途中で決めさせるようになっている。ビジネスコース系も充実している。
- 講義は基礎理論から実務に結びつく演習もある。私は Linuxサーバ構築演習 あたりが弱かったのでいい勉強になりました。
- 学生優遇が豊富（図書貸出・コワーキングスペースなど、実物とオンライン両方）。
  - 社会人でも学生系イベントに出られ、GitHub / Google アカウントを Pro 相当で使える。
- 卒業後も最新講義をほぼすべて聴講でき、学び直しが容易。
- 卒業研究の比重アップや講義アップデートなど改善が頻繁。
- 授業料が安い。編入なら卒業まで約130万円。フリーランス時代は全額経費計上で負担を最小化できた。
  - 大学中退の人も教養・外国語・一部専門基礎の単位認定が52までできるし、他にもIPOの資格は専門基礎の単位認定できました。
  - **来年から社会人学生の卒業まで授業料の50%相当額を免除制度が開始します。私の入学のときに欲しかった...**

## ここは弱いと感じた点
- 社会人学生の熱量がまちまちで、エンジニア志向が強い仲間は多くない。モラトリアム的受講者も一定数いて、公式コミュニティの技術ディスカッション密度にはばらつき。
- 実務レベルのものは学べる講義は少ない、Linuxサーバ構築演習
- 「サイバー大在学」と言うと舐められる場面がある。ただし逆に、成果を出したときにギャップで評価される面白さもある。

## 研究テーマとの接点
- 在学中に「AI検索は必ず伸びる」と確信し、卒業研究を AI検索のバイアス測定に設定。市場で波が来る前に枠組みを押さえることを狙った。
- 研究の詳細は別記事「AI検索サービスのバイアス監視システム（卒業研究）」を参照。
