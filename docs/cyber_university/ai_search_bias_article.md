# AI検索は本当に公平か？ — 企業優遇バイアスの定量測定に挑んだ卒業研究

## はじめに

この記事では、AI検索サービス（Perplexity AI）における「企業優遇バイアス」を定量的に測定・評価するシステムを構築し、市場競争への影響を分析した卒業研究についてご紹介します。

**論文情報**
- タイトル：AI検索サービスにおける企業優遇バイアス：市場競争への潜在的リスク
- 提出先：サイバー大学 IT総合学部
- 指導教員：田中 頼人 教授
- 提出日：2025年8月10日

---

## 第1章 研究の背景と目的

### 1.1 AI検索サービスの市場動向

ChatGPT、Perplexity AI、Microsoft Copilot、Google Geminiなど、生成AIベースの検索サービスが急速に普及しています。2025年6月時点では：

- Googleは依然として検索エンジン市場の89.54%を支配しています
- 生成AI市場ではChatGPTが82.24%のシェアを持ち、Perplexity（6.18%）、Gemini（5.33%）が続いています
- ガートナーは2026年までに従来の検索エンジンの検索ボリュームが25%減少すると予測しています

### 1.2 従来の検索エンジンとの違い

従来の検索エンジンでは複数の検索結果が並列的に提示されるため、ユーザー自身で比較検討できました。しかし、AI検索サービスでは統合された「答え」を提供するため、回答に含まれるバイアスがユーザーの判断に直接影響を与えるリスクが高まっています。

### 1.3 研究の目的

この研究では、以下の4つの目的に取り組みました：

1. **企業優遇バイアスの定量的測定手法の開発** - 企業・サービス評価に特化した新しい評価基準を提案します
2. **比較実証分析** - Perplexity AIとGoogle検索を対象に、AI検索サービスと従来型検索エンジンのバイアスを明らかにします
3. **市場構造との相関分析** - HHI（ハーフィンダール・ハーシュマン指数）を組み込んだ公平性評価を行います
4. **日本語圏初の包括的分析** - 言語・文化的背景による差異も分析します

### 1.4 企業優遇バイアスとは？

AI検索システムが特定の企業・ブランド・サービスに対して一貫した優遇的または冷遇的な評価を示す現象のことです。感情評価、推薦度合い、検索結果の内容など、複数の側面で現れる可能性があります。

## 第2章 関連研究と理論的背景

### 2.1 既存のAIバイアス研究

既存研究の大部分は性別、人種、年齢、宗教などの**社会的属性バイアス**に焦点を当てており、企業優遇バイアスに特化した研究は限られていました。

### 2.2 主要な先行研究

| 研究領域 | 主要研究 | 概要 |
|------|------|------|
| 社会的属性バイアス | Bolukbasi et al. (2016) | Word2Vecが性別ステレオタイプを学習することを実証しました |
| 自己選好バイアス | Wataoka et al. (2024) | LLMが自身の出力に高いスコアを付与する傾向を発見しました |
| 標準化評価 | Parrish et al. (2022) | BBQデータセットによるバイアス評価ベンチマークを提案しました |
| 検索エンジン操作効果 | Epstein & Robertson (2015) | 検索結果の順序が投票行動に20%の影響を与えることを実証しました |
| ブランドバイアス | Kamruzzaman et al. (2024) | LLMが国や所得レベルごとに異なるブランド推薦傾向を示すことを発見しました |
| 金融センチメント | 中川ら (2024) | 企業名の有無による評価スコアの差を「企業固有バイアス」と定義しました |

### 2.3 研究ギャップ

先行研究を調査した結果、以下のギャップが見つかりました：

1. AI検索サービスに関する研究の不足
2. 企業優遇バイアスの体系的研究の欠如
3. 市場競争への影響に関する実証研究の不足
4. 標準化されたデータセット・評価手法の不在
5. 日本語圏でのバイアス研究の不足

### 3.1 開発環境

この研究のシステム開発は、ちょうど「バイブコーディング」という言葉が注目され始めた時期でした。実験的にAIコーディングアシスタント「Cursor」を活用し、プロンプトベースでの開発に挑戦しましたが、正直なところ効率的とは言いがたい部分もありました。コンテキストの限界から、何度も同じ問題にぶつかることも少なくありませんでした。それでも、AIと協働する開発スタイルの可能性と課題を体感できた貴重な経験でした。

### 3.2 手法上の革新点

この研究では、以下の3つの革新的なアプローチを採用しました：

1. **感情評価×ランキング分析を統合した多軸バイアス検出フレームワーク**
2. **HHI（ハーフィンダール・ハーシュマン指数）を組み込んだ市場集中度考慮の公平性評価**
3. **週次データセット生成パイプライン**による時系列比較と業界粒度の可視化

### 3.4 8段階データ収集プロセス

データ収集は以下の8段階で行います：

| 段階 | 処理内容 | 詳細 |
|:---:|--------|------|
| 1 | Perplexity感情評価 | カテゴリ・業界全体の10点満点評価と実名指定の10点満点評価を取得します |
| 2 | Perplexityランキング | 特定カテゴリにおける企業・サービスの推薦順位を取得します |
| 3 | Perplexity引用リンク | APIレスポンスから情報源のURLと説明文を記録します |
| 4 | Google検索データ | Google Custom Search APIで同一クエリの結果を取得します |
| 5 | Perplexity感情判定 | 引用リンクのポジティブ/ネガティブ判定を行います |
| 6 | Google感情判定 | 検索結果のポジティブ/ネガティブ判定を行います |
| 7 | データ統合 | 異なるAPIからのデータ形式を調整し、統一的データセットを構築します |
| 8 | バイアス分析 | 企業優遇バイアス指標を算出します |

## 第4章 評価指標と分析手法

### 4.1 基本バイアス指標

#### Raw Delta（Δ）：生の差分値
```
Δ = 業界の感情評価 - サービス・企業の感情評価
```
- 正の値 → 優遇バイアス（実名表示により感情評価が向上）
- 負の値 → 冷遇バイアス（実名表示により感情評価が悪化）

#### NBI（Normalized Bias Index）：正規化バイアス指標
```
NBI = Δ / カテゴリ内平均|Δ|
```
- カテゴリ内での相対的なバイアスの大きさを評価できます
- 異なる業界・規模の企業間での直接比較が可能になります

### 4.2 市場指標：HHI（ハーフィンダール・ハーシュマン指数）

```
HHI = Σ(sᵢ × 100)²
```
- N：対象企業数
- sᵢ：各企業の市場シェア（0〜1の割合）
- HHI ∈ [0, 10000] で大きいほど集中度が高くなります

**解釈基準**
- HHI < 1,500：非集中市場
- 1,500 ≤ HHI < 2,500：中程度集中市場
- HHI ≥ 2,500：高集中市場

### 4.3 統計的有意性と効果量

統計的な信頼性を確保するため、以下の指標を使用しました：

| 指標 | 手法 | 基準 |
|------|------|------|
| 統計的有意性 | 符号検定 | p < 0.05 |
| 効果量 | Cliff's Delta | \|δ\| < 0.147：小、0.147 ≤ \|δ\| < 0.33：中、\|δ\| ≥ 0.33：大 |

### 4.4 ランキング分析指標

ランキングの分析には以下の指標を使用しました：

- **平均順位**：複数回のランキング取得結果における順位の平均値
- **順位相関分析**：Pearson相関係数とSpearman相関係数
- **ランキング安定性指標**：実行間での順位変動の標準偏差を正規化

### 4.5 比較分析指標

プラットフォーム間の比較には以下を使用しました：

- **Kendall's Tau**：両プラットフォーム間の順位相関
- **RBO（Rank Biased Overlap）**：上位ランキングに重みを置いた重複度評価
- **Overlap Ratio**：共通ドメインの出現頻度

### 4.6 公平性評価指標

#### 企業レベル公平性スコア
企業を時価総額で分類（大企業：100兆円以上、中企業：10〜100兆円、小企業：10兆円未満）し、3つの要素で評価しました：
- 大企業対小企業格差（35%）
- 中企業対小企業格差（35%）
- 全企業のバイアス分散（30%）

#### サービスレベル公平性スコア
```
サービスレベル公平性スコア = 平均NBI公平性成分 × HHI市場構造補正係数 × 集中度リスク調整
```

**HHI市場構造補正係数**
- HHI < 1,500（非集中市場）：1.0
- 1,500 ≤ HHI < 2,500（中程度集中市場）：0.9
- HHI ≥ 2,500（高集中市場）：0.7

## 第5章 分析結果と示唆

### 5.1 地域バイアスパターン

分析の結果、地域によって明確なバイアスパターンが見られました：

| 地域 | バイアス傾向 | 代表例 |
|------|----------|--------|
| 米国企業 | 正のバイアス | Microsoft (NBI: 1.653), Google (NBI: 1.188) |
| 中国企業 | 負のバイアス | Baidu (NBI: -1.743) |
| 日本企業 | 強い正のバイアス | ソニー (NBI: 1.56), ソフトバンク (NBI: 1.27) |

### 5.2 業界別バイアスパターン

業界ごとに特徴的なパターンが見つかりました：

#### 動画ストリーミング業界
全企業が正のバイアスを示す特徴的なパターンでした：
- Netflix (NBI: 1.493) - 業界内最高
- Amazon Prime Video (NBI: 0.82)
- Disney+ (NBI: 0.867)

#### SNS業界
世代別の明確なバイアス差異が見られました：
- TikTok (NBI: 1.863) - 最高
- Instagram (NBI: 1.808)
- Twitter/X (NBI: 0.164) - 従来型は軽微

#### 検索エンジン業界
地域による格差が顕著でした：
- Google (NBI: 0.485) - 軽微な正
- Bing (NBI: -0.915) - 強い負
- Baidu (NBI: -1.743) - 非常に強い負

#### AI検索サービス業界
興味深い結果が得られました：
- Google AI Overviews (NBI: 1.354) - 最高
- Perplexity (NBI: 0.984)
- Microsoft Copilot (NBI: 0.554)
- ChatGPT (NBI: -1.108) - **競合関係によるバイアスの可能性があります**

### 5.3 感情評価とランキング分析の相関性

感情評価とランキングの間に、業界ごとに異なる相関が見られました：

| 業界 | 相関係数 | p値 | 解釈 |
|------|--------|-----|------|
| SNS | 0.964 | 0.036 | 高い一貫性 |
| 自動車メーカー | 0.915 | 0.029 | 高い一貫性 |
| 世界的テック企業 | 0.892 | 0.042 | 高い一貫性 |
| 日本の大学 | 0.732 | 0.007 | 高い一貫性 |
| AI検索サービス | -0.329 | - | **負の相関（感情と機能の乖離）** |

### 5.4 公平性評価結果

各業界の公平性スコアを算出した結果、以下のようになりました：

| 業界 | サービスレベル公平性スコア | 解釈 |
|------|------------------------|------|
| クラウドサービス | 0.48 | 比較的高い公平性 |
| SNS | 0.327 | 中程度 |
| 検索エンジン | 0.149 | 深刻な市場支配力格差 |
| 動画ストリーミング | 0.123 | 低い |
| AI検索サービス | 0.063 | 極めて低い |
| オンラインショッピング | 0.019 | **最も深刻** |

### 5.5 重要な発見

この研究を通じて、以下の重要な発見がありました：

1. **競合関係によるバイアス**: PerplexityがGoogle検索を相対的に低く評価（NBI: 0.485）し、ChatGPTに負のバイアス（NBI: -1.108）を示していました
2. **内部動作の検証**: 架空サービス"Hoge Cloud"（NBI: -3.136）への極端な負のバイアスは、Perplexityが企業の実在性を検証する機能を持つことを示唆しています
3. **感情評価と機能評価の乖離**: AI検索サービス業界のみ負の相関を示しました

## 第6章 結論と今後の課題

### 6.1 主要な研究成果

この研究では、以下の成果を得ることができました：

1. **多元的要因によるバイアス構造の実証**: 地域性、業界特性、競合関係などが影響することを明らかにしました
2. **地域バイアスの発見**: 米国企業優遇、中国企業冷遇、日本企業優遇というパターンを発見しました
3. **業界固有パターンの特定**: 動画ストリーミング（全企業優遇）、SNS（世代別格差）などを特定しました
4. **感情-ランキング相関の発見**: SNS（0.964）、自動車（0.915）で高い一貫性があることを確認しました
5. **競合関係バイアスの実証**: Perplexityが競合（ChatGPT、Google）を低評価していることを実証しました

### 6.2 統計的信頼性

研究の信頼性を確保するため、以下の条件で分析を行いました：

- 実行回数：15回
- 安定性スコア：0.94以上
- 分析対象企業の大部分で統計的有意性（p < 0.05）と大きな効果量（|Cliff's Delta| > 0.474）を確認しました

### 6.3 研究の限界

この研究には以下の限界があります：

- 日本語での検索・回答に基づく言語的バイアスの可能性
- 2025年8月3日時点でのスナップショット分析
- 分析対象が事前定義された企業・サービスに限定

### 6.4 今後の課題

今後は以下の方向性で研究を発展させたいと考えています：

1. **研究手法の発展**: Perplexity以外のAI検索サービスへの拡張
2. **政策・実務応用**: 競争政策への活用、AI-SEO手法の開発

## おわりに

今回の研究では、AI検索サービスにもバイアスが存在し、競合サービスを低評価する傾向すらあることが分かりました。

そしてこの研究の後、AIエージェントの時代が急速に到来しています。ChatGPTのショッピングアシスタント機能のように、AIが直接商品を提案・購入まで導く世界が現実になりつつあります。「検索結果を見せる」から「AIが選んで提案する」へ——バイアスの影響はより直接的になるでしょう。今後は計測方法や影響度の評価も、この変化に合わせてアップデートしていく必要があると感じています。

AI検索の公平性について考えるきっかけになれば幸いです。ご質問やご意見がありましたら、お気軽にお声がけください。
