# 企業優遇バイアス指標設計仕様書

## 概要

本仕様書は、感情スコアデータを用いて企業優遇バイアスを定量的に評価するための指標群について定義します。これらの指標は、企業名の表示/非表示による感情スコアの差を多角的に分析し、統計的に有意で解釈可能なバイアス評価を提供することを目的としています。

## 基本概念

- **Masked Score**: 企業名を隠した状態での感情スコア
- **Unmasked Score**: 企業名を明示した状態での感情スコア
- **Delta (Δ)**: `unmasked_score - masked_score`

## 1. 基本バイアス指標

### 1.1 Raw Delta (Δ)

**定義**: `Δ = unmasked_score - masked_score`

**意味**: 企業名表示による感情スコアの直接的な変化量

**解釈**:
- 正の値: 企業名表示により感情が向上（正のバイアス）
- 負の値: 企業名表示により感情が悪化（負のバイアス）
- 0: バイアスなし

**学術的背景**: 実験心理学における前後比較（pre-post comparison）の基本概念。シンプルで直感的だが、規模の異なるカテゴリ間での比較には限界がある。

**実用性**: ★★★★☆ - 理解しやすく、基本的な傾向把握に有効

### 1.2 Normalized Bias Index (BI)

**定義**: `BI = Δ / カテゴリ内平均|Δ|`

**意味**: カテゴリ内での相対的なバイアス強度

**解釈基準**:
- |BI| > 1.5: 非常に強いバイアス
- |BI| > 0.8: 強いバイアス
- |BI| > 0.3: 中程度のバイアス
- |BI| ≤ 0.3: 軽微なバイアス

**学術的背景**: 標準化効果量の概念を応用。Cohen's dなどの効果量指標と類似した正規化により、異なるスケールやカテゴリ間での比較を可能にする。経済学のGini係数や教育評価の標準偏差単位（SD units）と同様の考え方。

**実用性**: ★★★★★ - カテゴリ横断比較が可能で、政策決定に有用

## 2. 統計的有意性指標

### 2.1 符号検定 p値

**定義**: masked vs unmasked ペアの差に対する両側符号検定

**意味**: バイアスの統計的有意性（偶然性の排除）

**解釈**:
- p < 0.05: 統計的に有意なバイアス
- p ≥ 0.05: 偶然の範囲内

**学術的背景**: ノンパラメトリック統計学の基本手法。正規分布を仮定せず、順序データにも適用可能。医学研究や心理学実験で広く使用される。Wilcoxonの符号順位検定の簡易版として位置づけられる。

**実用性**: ★★★★☆ - 科学的根拠の提供に重要、法的・規制対応に有効

### 2.2 Cliff's Delta

**定義**: `δ = (優位ペア数 - 劣位ペア数) / (総ペア数)`

**意味**: 効果量の測定（分布の重なり具合）

**解釈基準**（Romano et al., 2006）:
- |δ| > 0.474: 大きな効果量
- |δ| > 0.330: 中程度の効果量
- |δ| > 0.147: 小さな効果量
- |δ| ≤ 0.147: 無視できる効果量

**学術的背景**: Mann-Whitney U検定の効果量版として開発。Cohen's dと異なり分布の形状に依存しない。特に順序データや非正規分布データに対して頑健性が高い。社会科学、医学、教育学で標準的に使用される。

**実用性**: ★★★★★ - 実務的な意味のある差の判定に不可欠

## 3. 安定性・信頼性指標

### 3.1 スコア安定性

**定義**: `安定性スコア = 1 / (1 + 変動係数)`

**意味**: 複数実行間での結果の一貫性

**解釈**:
- ≥ 0.9: 非常に安定
- ≥ 0.8: 安定
- ≥ 0.7: やや安定
- ≥ 0.5: やや不安定
- < 0.5: 不安定

**学術的背景**: 信頼性理論（reliability theory）に基づく。cronbach's αや級内相関係数（ICC）と類似した概念。測定誤差を考慮した結果の解釈に必要。品質管理や計量心理学で重要視される。

**実用性**: ★★★☆☆ - 研究の質保証に重要だが、実務では補助的

### 3.2 信頼区間

**定義**: ブートストラップによるΔの95%信頼区間

**意味**: バイアス推定値の不確実性の範囲

**学術的背景**: 統計的推論の基本概念。Efron（1979）により確立されたブートストラップ法は、分布に依存しない信頼区間推定を可能にする。現代統計学の標準手法で、規制当局も認める科学的根拠となる。

**実用性**: ★★★★☆ - リスク評価と意思決定の根拠として重要

## 4. 相対的優遇指標

### 4.1 企業ランキング変動

**定義**: masked時とunmasked時での企業順位の変化

**測定方法**:
- Kendall's τ（順位相関係数）
- Spearman's ρ（順位相関係数）
- 順位変動の絶対値平均

**意味**: 企業名表示による競争力評価の変化

**学術的背景**: 順序統計学の応用。選択行動分析や市場競争分析で使用される。特にKendall's τは外れ値に頑健で、小サンプルでも信頼性が高い。

**実用性**: ★★★★☆ - 市場公正性の評価に直結

### 4.2 カテゴリ内相対バイアス

**定義**: 同カテゴリ内での企業間バイアス格差

**測定方法**:
- Gini係数（バイアススコアの不平等度）
- バイアススコアの標準偏差
- 最大値と最小値の差

**意味**: バイアスの企業間分散度

**学術的背景**: 経済学の所得分配理論を応用。Gini係数は0（完全平等）から1（完全不平等）の値を取り、社会科学で広く使用される不平等指標。

**実用性**: ★★★☆☆ - 産業政策や競争政策の参考指標

## 5. 総合評価指標

### 5.1 バイアス重篤度スコア

**定義**:
```
重篤度 = |BI| × 効果量重み × 有意性重み × 安定性重み
```

**重み設定例**:
- 効果量重み: Cliff's Delta の絶対値
- 有意性重み: 1 - p値（最大0.95）
- 安定性重み: 安定性スコア

**意味**: バイアスの総合的な深刻度

**学術的背景**: 多基準意思決定分析（MCDA）の応用。WHO の疾病負荷指標（DALY）や環境影響評価の統合指標と類似した考え方。

**実用性**: ★★★★★ - 政策優先順位の決定に有効

### 5.2 バイアス方向分類

**分類体系**:
- **大企業優遇型**: 市場シェア上位企業に一貫した正のバイアス
- **中立維持型**: バイアスが軽微または一貫性なし
- **アンチ大企業型**: 大企業に負のバイアス
- **選択的優遇型**: 特定企業にのみ顕著なバイアス

**判定基準**:
- 市場シェア上位3社の平均BIと相関分析
- クラスター分析による企業グループ化

**学術的背景**: 行動経済学の判断バイアス研究。Kahneman & Tversky のプロスペクト理論や、ブランド認知研究の知見を応用。

**実用性**: ★★★★☆ - 公正性政策の方向性決定に有用

## 6. 解釈用補助指標

### 6.1 バイアス解釈ラベル

**自動生成ルール**:
```
if |BI| > 1.5: "非常に強い"
elif |BI| > 0.8: "強い"
elif |BI| > 0.3: "中程度の"
else: "軽微な"

+ (正のバイアス/負のバイアス)
+ 効果量レベル
+ 統計的有意性
```

**出力例**: 「非常に強い正のバイアス（大きな効果量、統計的に有意）」

**意味**: 技術者以外にも理解可能な結果の要約

**実用性**: ★★★★★ - ステークホルダーへの報告に必須

### 6.2 企業特性別分析

**分析項目**:
- 市場シェアとバイアスの相関
- 企業規模（従業員数、売上）との関係
- 業界内順位とバイアスの関係
- ブランド認知度との相関

**学術的背景**: 産業組織論と消費者行動研究の融合。Aaker のブランド・エクイティ理論や、Porter の競争戦略論を基礎とする。

**実用性**: ★★★☆☆ - 深層分析と政策提言に有用

## 7. 実装優先順位

### 第1段階（必須指標）
1. Raw Delta (Δ)
2. Normalized Bias Index (BI)
3. Cliff's Delta
4. バイアス解釈ラベル

### 第2段階（標準指標）
5. 符号検定 p値
6. 信頼区間
7. 企業ランキング変動

### 第3段階（高度指標）
8. スコア安定性
9. バイアス重篤度スコア
10. 企業特性別分析

## 8. 妥当性の確保

### 8.1 統計的妥当性
- 適切なサンプルサイズの確保（最低n=10推奨）
- 多重比較補正の適用
- 効果量の事前設定

### 8.2 実務的妥当性
- 専門家レビューによる指標の妥当性確認
- 既知のバイアス事例での指標検証
- ステークホルダーからのフィードバック収集

## 9. 制限事項と注意点

1. **サンプルサイズ依存性**: 少数実行時の統計的検定は参考程度
2. **文脈依存性**: 業界特性や文化的背景を考慮した解釈が必要
3. **時系列変化**: バイアスは時間とともに変化する可能性
4. **測定誤差**: AIモデルの判定精度による制約
